## Setup and Installation

1. Clone repo

```
git clone git@github.com:zultopia/HAP_BoloBoloBoys.git
```

2. Compile the program

--- Isi ---

### Linux

Buka terminal pada folder project, buat folder bernama bin, lalu jalankan makefile dengan perintah

```
make clean
make run
```
